## Description

This module provides a Redis store for [Aclify](https://github.com/Aclify/aclify) package.

## Install

Npm:
```bash
$ npm install @aclify/redis --save
```

Yarn:
```bash
$ yarn add @aclify/redis
```

## Tests
```bash
    $ npm run docker:run:test
```
